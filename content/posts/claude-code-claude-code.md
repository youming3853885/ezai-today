# Claude Code 新手從零開始攻略 |                             什麼是claude code?

* * *

## 1.Claude Code是啥? 與GPT有啥差別?

2026年的今日，如果要我只推一個當今最好用的vibe coding工具，大概就只會是claude code (以下簡稱CC)。簡單來說，它就是一個可以常駐在電腦裡面的全能助手，舉凡開發APP、寫網頁、搞行銷、資料蒐集、做簡報、製作遊戲，幾乎任何能從電腦產出的東西，它都能做。

與chat gpt最大的差別在於，gpt像是個學識淵博的教授，沒有什麼東西難的倒他!但他只能永遠躲在房間(雲端)內，透過門縫伸手交出你要的資料。你電腦上任何疑難雜症，它都看不到，也幫不了你。要求助這位老教授，就只能靠你透過門縫，主動傳遞你需要幫忙的事情，它會給建議，但你還是要自己動手做。

而CC，像是永遠住在你電腦裡工程師夥伴，可以看的到你寫的報告，你寫壞的程式，你開發到一半的遊戲。當你累了或卡住時，它能夠直接幫你繼續接手完成，而我們只需要泡杯咖啡等它工作完再來驗收成果就好。

## 2.學習Claude Code有什麼好處？

1.  **把想法化為現實。**
    
    做個相片分類+濾鏡APP? 網頁互動小遊戲? 公司下午茶團購系統?。如果你是腦袋有很多想法的人，但苦於傳統寫程式門檻高，以前覺得「不可能自己做」的東西，現在透過CC是真的可以開發出來。雖然APP store就有很多功能型APP可以解決日常生活大小事，但總是有些"客製化”需求沒法被滿足，而CC剛好能補足這個缺口。只要有想法，就有可能實踐。
    
2.  **提升效率。**
    
    工作中最討厭的就是「重複再重複」的事，對吧？像是：
    
    -   每月整理報表：從不同Excel拉資料、算總和、做圖表。
    -   客戶資料清理：名字格式不統一、電話有空格、重複email。
    -   寫重複的email或報告。
    
    以前這些事要花半天，現在可直接跟Claude Code說：「生成某月報表、幫我把高雄區的客戶寄送一份新廣告企劃。」它就直接動手，讀檔、處理資料、寄送、一氣呵成。
    
3.  **副業幫手。**
    
    由於CC速度快，成本極低，YT/threads上不乏看到很多非資訊出身的人，利用CC接案，知識販售，作內容創作+個人品牌，或是創立SaaS服務等數位交付的產品。
    
    總結:不需要變成超厲害的工程師，而是學會後讓你從「想做但不會」變成「**想做就做**」。

# 3.學習門檻

跟傳統學程式語言相比，CC的門檻真的是低太多的太多。程式語言需要懂邏輯、框架、語法。而vibe coding最大的優點就是，只要會說話就好!只要能夠清楚表達創作內容，即可驅動CC幫你動工。

(但這裡有一個門檻: 如何**清楚**的表達意圖?之後這系列也會分享我踩過的各種坑)

而學CC唯一的門檻是…….你需要有一顆不怕犯錯的心，跟不斷測試的動力。它很強大，但需要清楚的指令，它很快速，但需要合適的規範，接下來，這系列會分享CC從安裝到產品開發再到自動化的全流程。
