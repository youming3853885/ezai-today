# 公眾號 Markdown 編輯器

歡迎使用這款專為**微信公眾號**設計的 Markdown 編輯器！✨

## 🎯 核心功能

### 1. 智慧圖片處理

![](https://images.unsplash.com/photo-1618005198919-d3d4b5a92ead?w=800&h=500&fit=crop)

- **貼上即用**：支援從任何地方複製貼上圖片（截圖、瀏覽器、檔案管理器）
- **自動壓縮**：圖片自動壓縮，平均壓縮 50%-80%
- **本地儲存**：使用 IndexedDB 持久化，重新整理不丟失
- **編輯流暢**：編輯器中使用短連結，告別卡頓

### 2. 多圖排版展示

支援朋友圈式的多圖網格佈局，2-3 列自動排版：

![](https://images.unsplash.com/photo-1498050108023-c5249f4df085?w=600&h=400&fit=crop)
![](https://images.unsplash.com/photo-1517694712202-14dd9538aa97?w=600&h=400&fit=crop)
![](https://images.unsplash.com/photo-1461749280684-dccba630e2f6?w=600&h=400&fit=crop)

### 3. 13 種精美樣式

1. **經典公眾號系列**：預設、技術、優雅、深度閱讀
2. **傳統媒體系列**：雜誌、紐約時報、金融時報、Jony Ive
3. **現代數字系列**：Wired、Medium、Apple、Claude、AI Coder

### 4. 一鍵複製

點選「複製到公眾號」按鈕，直接貼上到公眾號後臺，格式完美保留！

## 💻 程式碼示例

```javascript
// 圖片自動壓縮並儲存到 IndexedDB
const compressedBlob = await imageCompressor.compress(file);
await imageStore.saveImage(imageId, compressedBlob);

// 編輯器中插入短連結
const markdown = `![圖片](img://${imageId})`;
```
